
# 커플 대시보드 (Couple Dashboard) 💖

재훈 ❤️ 도영 커플을 위한 개인화된 대시보드 웹앱입니다.

## 🚀 주요 기능

- **실시간 헤더**: 커플 이름과 함께 현재 날짜 및 시간을 실시간으로 표시합니다.
- **기념일 카운터 (D-Day)**: 중요한 날짜를 등록하고 D-Day(D+, D-)를 자동으로 계산합니다. 상세한 날짜(년, 월, 일) 계산 기능이 포함되어 있습니다.
- **개인 정보 카드**: 커플의 이름, 생년월일, 주소, MBTI, 전화번호를 한눈에 확인할 수 있습니다.
- **오늘의 운세**: MBTI 기반으로 매일 업데이트되는 사랑운, 금전운, 건강운 및 행운의 아이템을 제공합니다.
- **추천 데이트 코스**: 날씨(실내/야외)에 따른 서울 핫플레이스 데이트 코스를 추천합니다.
- **공유 일정**: 다가오는 일정을 리스트 형태로 관리합니다. 날짜순으로 자동 정렬됩니다.
- **실시간 동기화**: Vercel KV를 사용하여 기기 간 실시간으로 데이터를 동기화합니다.

## 🛠 기술 스택

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Database**: Vercel KV (Redis) - 실시간 데이터 저장 및 동기화

## 🎨 디자인 컨셉

- **Rose/Pink Theme**: 사랑스러운 분위기를 위해 로즈 톤의 색감과 부드러운 애니메이션을 적용했습니다.
- **Responsive Design**: 모바일과 데스크톱 모두에 최적화된 반응형 레이아웃을 제공합니다.
- **Glassmorphism**: 부드러운 블러 효과와 카드 UI를 사용하여 세련된 느낌을 줍니다.

## 📦 설치 및 실행

1. 의존성 설치:

```bash
npm install
```

1. 환경 변수 설정:
`.env.local` 파일을 생성하고 Vercel KV 설정을 추가합니다.

2. 개발 서버 실행:

```bash
npm run dev
```

1. 브라우저에서 `http://localhost:3000` 접속

## 📂 프로젝트 구조

- `app/`: 페이지 구성 및 API 엔드포인트
- `components/`: 대시보드를 구성하는 각 기능별 컴포넌트
- `lib/`: KV 저장소 설정 및 데이터 타입 정의
- `public/`: 정적 파일
